# 🔐 React Auth System (Clean UI)

## 📌 Overview
This project is a clean and minimal authentication system built using React.  
It includes Signup, Login, and Dashboard functionality with proper validation and protected routing.

---

## 🚀 Features
- User Signup with username and email
- Login authentication system
- Multi-user support using localStorage
- Protected dashboard route
- Logout functionality
- Light and Dark mode support
- Clean and professional UI design

---

## 🛠 Technologies Used
- React.js
- React Router DOM
- CSS
- LocalStorage
